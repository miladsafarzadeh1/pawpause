Stop your cat from typing in your app.

Paw Pause detects when a cat walks on the keyboard and suppresses the input — before it posts gibberish to a channel, fires a destructive shortcut, or mangles a draft. Humans keep typing normally; cats get clamped.

THE MODEL

How detection works

Every keystroke updates a rolling ~1.5 second window. Six interpretable signals sum into a score; cross the threshold and input is suppressed. A human trips maybe one signal incidentally; a paw-flop trips several at once. No machine learning, no keystroke logging — only physical key codes are inspected, never the characters typed.

SIGNALFIRES WHENWEIGHT
Concurrent keys3+ keys physically held at once0.7
Heavy press5+ keys held — sitting on it0.7
Burst rate8+ keys / second0.4
Adjacency roll3+ consecutive neighboring keys0.4
Repetitionsame key 6+ times0.3
No structure6+ keys, no space / return / delete0.2

Default trigger threshold: 1.0. Every weight and threshold is configurable. A false positive never traps a real user — auto-release is the default and Escape always unlocks.

QUICKSTART

Drop it into any text field

VANILLA — UNCONTROLLED

import { attachGuard } from "pawpause";

const el = document.querySelector("#composer");
const guard = attachGuard(el, {
  onClamp: (i) => toast("Held cat input"),
  onRelease: () => toast("All clear"),
});
// guard.disarm(); guard.destroy();

REACT — CONTROLLED (SLACK-STYLE)

import { usePawPause } from "pawpause/react";

const { ref, blocking, lastClamp } =
  usePawPause({ rollback: false });

useEffect(() => {
  if (lastClamp) setValue(lastClamp.safeValue);
}, [lastClamp]);

<textarea ref={ref} value={value} ... />
<button disabled={blocking}>Send</button>
